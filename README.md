# Employee Retention Predictive Models for HR Groups

## 📌 Project Overview

This project focuses on predicting employee attrition using IBM SPSS Modeler by analyzing various employee-related factors that influence retention and turnover. The objective of this project is to apply predictive analytics and machine learning techniques to identify employees who are likely to leave the organization.

Using IBM SPSS Modeler, the project performs data preprocessing, analysis, and predictive modeling to understand patterns in employee behavior and retention trends. This project was developed as part of an IBM academic subject to gain practical exposure to predictive analytics and HR data analysis.

---

# ❓ Problem Statement

Organizations often face challenges in identifying employees who may leave the company. Traditional HR methods detect attrition only after employees resign, making it difficult to take preventive actions.

The problem addressed in this project is:

**How can employee attrition be predicted using employee-related data to identify turnover trends and support employee retention strategies?**

Predicting employee attrition in advance helps HR departments improve workforce management, reduce turnover costs, and support employee satisfaction.

---

# 🎯 Objectives

The primary objectives of this project are:

- To analyze employee-related data using IBM SPSS Modeler
- To identify factors affecting employee attrition
- To build a predictive model for employee retention analysis

# ⚙️ Methodology

The project follows a structured predictive analytics workflow.

## Data Collection

Employee data was collected and organized into a structured format suitable for analysis.

## Data Preprocessing

The dataset was cleaned and prepared by:

- Removing missing values
- Handling inconsistent records
- Formatting data properly
- Selecting relevant attributes

## Data Analysis

Exploratory analysis was performed to understand relationships between variables affecting employee attrition.

## Model Building in IBM SPSS

A predictive model was created using IBM SPSS Modeler Stream to analyze employee retention patterns.

The process included:

- Importing dataset
- Data transformation
- Variable selection
- Statistical analysis
- Model training
- Prediction generation

## Result Evaluation

The prediction results were analyzed to determine model effectiveness and attrition trends.

# 📈 Results

The predictive model successfully analyzed employee-related factors and generated attrition prediction outcomes.

## Key Findings

- Overtime significantly affects employee attrition
- Job satisfaction influences employee retention
- Monthly income impacts turnover patterns
- Multiple HR indicators improve prediction accuracy

The model demonstrated how HR analytics can support workforce management and employee retention strategies.
