# Employee Retention Predictive Models for HR Groups

## Overview
This project predicts employee attrition using machine learning techniques in IBM SPSS Modeler.

## Technologies Used
- IBM SPSS Modeler
- Machine Learning
- Predictive Analytics

## Nodes Used
- Var File Node
- Type Node
- Filter Node
- Select Node
- Modeling Node
- Evaluation Node
- Table Node

## Dataset Features
- Age
- Department
- Job Role
- Monthly Income
- Overtime
- Work Experience
- Job Satisfaction
- Attrition Status

## Objective
To help HR departments identify employees who are likely to leave the organization and improve employee retention strategies.

## Outcome
The model analyzes employee data and predicts attrition patterns for better workforce management.
